# Kakuro Game

A Java implementation of the **Kakuro puzzle game** with an integrated constraint-based solver, built during the **4th semester (Spring 2022) of B.Sc. in Computer Science at the University of Strasbourg**.

## Technical Highlights

- **Language:** Java with OOP design — encapsulation, inheritance, and polymorphism
- **Solver:** constraint satisfaction / backtracking algorithm that finds valid solutions for any Kakuro grid
- **GUI:** graphical interface for interactive gameplay — cell selection, number input, and validation
- **Game logic:** sum constraints across rows and columns with unique-digit rules, full rule enforcement and win-state detection
- **Project structure:** organized under `Kakuro_project/` with separation of game logic, solver, and UI components
